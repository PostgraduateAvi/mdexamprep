1. Ominous Octet 3 marks — name all 8
DeFronzo's 8 pathogenic mechanisms in T2DM. PROPER-NOUN GATE — naming all 8 is a zero-or-ten differentiator.
1Decreased insulin secretion — β-cell failure (progressive, starts before diagnosis, ~50% function lost at diagnosis)
2Increased glucagon secretion — α-cell dysfunction, non-suppressed glucagon → hepatic glucose output
3Increased hepatic glucose production — gluconeogenesis + glycogenolysis, driven by insulin resistance + excess glucagon
4Decreased muscle glucose uptake — skeletal muscle insulin resistance (GLUT4 downregulation), largest glucose disposal site
5Increased lipolysis — adipocyte insulin resistance → ↑FFA → lipotoxicity → worsens hepatic & muscle IR
6Decreased incretin effect — GLP-1 and GIP impaired → basis for GLP-1 RA and DPP-4 inhibitor therapy
7Increased renal glucose reabsorption — SGLT2 upregulated in proximal tubule → basis for SGLT2 inhibitors
8Neurotransmitter dysfunction — hypothalamic appetite dysregulation, dopamine pathway → basis for bromocriptine use
📝 Each of the 8 maps to a drug class. The examiner loves: "Name the octet and the drug targeting each." This is a 3-mark question answered in a table.
2. Insulin Resistance Mechanism 1.5 marks
Normal: Insulin binds α-subunit of insulin receptor (tyrosine kinase) → autophosphorylation of β-subunit → IRS-1 phosphorylation → PI3K → GLUT4 translocation to membrane → glucose uptake.

In T2DM: Post-receptor defect — serine phosphorylation of IRS-1 (instead of tyrosine) → blocks PI3K pathway → GLUT4 stays intracellular. Driven by: FFA (lipotoxicity), inflammatory cytokines (TNF-α, IL-6 from visceral adipose), ER stress.

Insulin receptor was asked directly in 2007. Key: it's a heterotetrameric receptor (2α + 2β subunits) with intrinsic tyrosine kinase activity on the β-subunit.
📝 One sentence answer: "Post-receptor defect with serine (not tyrosine) phosphorylation of IRS-1 blocking GLUT4 translocation." This single sentence scores the mechanism mark.
3. β-Cell Failure 1 mark
Progressive. ~50% β-cell function lost at diagnosis. Mechanisms: glucotoxicity (chronic hyperglycemia → oxidative stress → β-cell apoptosis), lipotoxicity (FFA → ceramide → apoptosis), amyloid deposition (IAPP/amylin aggregates in islets), genetic susceptibility (TCF7L2 strongest T2DM gene).
📝 "50% function lost at diagnosis" + "glucotoxicity + lipotoxicity + amyloid" = complete answer. TCF7L2 is the proper-noun gate if genetics is asked.
4. Diagnostic Criteria 1 mark
ADA criteria (any one, confirmed on repeat): FPG ≥126 mg/dL · 2h OGTT ≥200 mg/dL · HbA1c ≥6.5% · Random glucose ≥200 + symptoms

Prediabetes: FPG 100-125 (IFG) · 2h OGTT 140-199 (IGT) · HbA1c 5.7-6.4%

GDM screening (asked 3× in PI): IADPSG/WHO one-step 75g OGTT at 24-28 weeks. Thresholds: FPG ≥92, 1h ≥180, 2h ≥153. One abnormal value = diagnosis.
📝 GDM numbers (92/180/153) are proper-noun gates — asked 3 times. Must memorize exactly.
5. Octet → Drug Map 1.5 marks (if management asked)
1β-cell failure → Sulfonylureas, Meglitinides (insulin secretagogues)
2α-cell glucagon → GLP-1 RA, DPP-4i (suppress glucagon)
3Hepatic glucose → Metformin (↓hepatic gluconeogenesis via AMPK)
4Muscle IR → Thiazolidinediones (PPARγ → ↑insulin sensitivity)
5Lipolysis → TZDs (also adipocyte)
6Incretin defect → GLP-1 RA (exenatide, liraglutide, semaglutide), DPP-4i (sitagliptin)
7Renal reabsorption → SGLT2i (empagliflozin, dapagliflozin)
8Brain dysfunction → Bromocriptine (D2 agonist)
